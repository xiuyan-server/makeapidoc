    	// {action} 会自动换成对应的方法名

## 如何使用  

### 1.放置文件
	如果你是Thinkphp5，则将 extend 目录里的 mumbaicat 放在 TP5 的 extend 目录下 。

### 2.引入
    use mumbaicat\makeapidoc\ApiDoc;

### 3.在合适地方实例化
	$doc = new ApiDoc('../application');
	// 参数1是代码目录，参数2是保存路径，参数2默认是当前路径。 注意斜杠,windows是/ ，Linux/Mac是\ ，建议使用PHP常常量 DIRECTORY_SEPARATOR
	echo $doc->make();
